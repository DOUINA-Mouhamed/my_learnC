## How it works

You have multiple directories in this repo. First you will have to clone it using the ```git clone``` command. Secondly, you can see that there is an empty directory called ```my_mini_projects```. You will create your C files here when you arrive at the ```exercises``` step üíª.

* Step 3 : You have finished your mini-projects, and now you can finally step into the ```my_learnC/02_statements``` directory.
